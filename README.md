# あみだくじツール
JAWS-UG 初心者支部 #74 で石原さんからご提供いただいた [2025 jaws-ug 紹介本 (初心者支部有志)](https://github.com/takano-suzu69/2025jawsugbgnrbook) の物理本を配布するために、あみだくじツールを KIRO で作りました。

## 機能

- 参加者リストを貼り付けて簡単に抽選
- 当選者数を自由に設定可能
- アニメーション付きのあみだくじ演出
- レスポンシブデザイン対応
- ダークテーマUI

## 使い方

1. `index.html` をブラウザで開く
2. 参加者名を1行に1名ずつ貼り付け
3. 当選者数を入力（デフォルト: 5名）
4. 「抽選へ」ボタンをクリック
5. 「スタート」ボタンで抽選開始
6. アニメーション終了後、「結果発表」ボタンで当選者を表示

## 注意事項

- 参加者数は当選者数以上である必要があります
- 重複した名前は自動的に除外されます
- ブラウザで直接開くだけで動作します（サーバー不要）

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
